### inicializar entorno virtual

```bash
python -m venv .venv
```

### Activar entorno virtual

- Linux/MacOS
  
  ```bash
  source .venv/bin/activate
  ```
  
- Windows (pf use windows terminal y powershell)
  
  ```powershell
  .\.venv\Scripts\activate.ps1
  ```

### Instalar dependencies

```bash
pip install -r requirements.txt
```
